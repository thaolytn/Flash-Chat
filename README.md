# Flash-Chat
An iOS project that simulates a simple, two-user messaging app.

## Tech Stacks
- 🔨 Swift 5
- 🎨 UIKit
- 🍫 CLTypingLabel
- 🪟 MVC
- ☁️ Firebase
- 

## Setup
1. To run this project, install it locally using Terminal...
```
$ git clone https://github.com/thaolytn/Flash-Chat.git
$ open Flash-Chat.xcworkspace
```
